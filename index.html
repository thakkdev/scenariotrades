ScenarioTrades

A custom rule‑based alert engine. Define your logic — ScenarioTrades runs it and notifies you when conditions are met.

What is ScenarioTrades?

ScenarioTrades is a modular alert engine that evaluates user‑defined rules against market data and sends alerts when conditions are met. It does not provide trading advice, does not recommend trades, and does not execute trades. It simply runs the logic you define.

How It Works

  • You define a rule using your own logic.
  • ScenarioTrades continuously runs that rule logic.
  • When your conditions are met, an alert is sent.
  • No optimization, no interpretation — your logic, your conditions.

Supported Modules

Rules can reference any combination of:

  • Price
  • Volume
  • Options flow
  • Sentiment
  • Markov state transitions
  • Custom data sources (user‑provided)

Modules are optional — you choose what your rule uses.

Rule Logic Examples

Price + Volume Example

IF price crosses above 20‑day SMA AND volume > 2× average THEN send alert

Sentiment + Flow Example

IF sentiment flips from negative → positive AND call‑volume > put‑volume THEN send alert

Markov Chain Example (Non‑Predictive)

IF Markov state transitions from S2 → S3 THEN send alert

These examples demonstrate logic only — not performance, predictions, or trading outcomes.

What ScenarioTrades Does Not Do

  • Does not recommend trades
  • Does not optimize rules
  • Does not provide performance claims
  • Does not execute trades
  • Does not manage money
